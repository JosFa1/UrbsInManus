────────────────────────────────────────────────────────────────────
NOTES ON ASSET CREATION
────────────────────────────────────────────────────────────────────

1. PIXEL PERFECT: All assets should align to 48-pixel grid units
   - 1x1 = 48x48px
   - 2x2 = 96x96px
   - 3x3 = 144x144px
   - etc.

2. TRANSPARENCY: Use PNG-24 with alpha channel for all buildings
   
3. SHADOWS: Add subtle drop shadows beneath buildings for depth

4. STYLE: Keep a consistent art style across all assets
   - Suggested: Isometric 2D or top-down 2.5D perspective
   - Alternative: Simple flat design with strong outlines

5. COLOR PALETTE (suggested):
   - Stone: #A8978E, #D3C5BB
   - Terracotta: #CD853F, #D2691E
   - Marble: #F5F5DC, #FFFAF0
   - Roof tiles: #8B4513, #A0522D
   - Grass: #6B8E23, #556B2F
   - Water: #4682B4, #5F9EA0

6. NAMING: Use lowercase, no spaces, match exactly as listed above

7. FILE STRUCTURE:
   /graphics/
     buildings/
       via.png
       insula.png
       domus.png
       [etc...]
     ui/
       cursor-normal.png
       icon-water.png
       [etc...]
     terrain/
       terrain-grass.png
       overlay-green.png
       [etc...]
     effects/
       effect-fire.png
       [etc...]
